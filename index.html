Difference between null and undefined

Null:

Null means it's empty i.e অস্তিত্ব আছে কিন্তু খালি। null যদি কোন variable এ assign করা থাকে তাহলে পরবরতিতে সেটা change করা যায়।


Undefined:

Undefined means it's not defined i.e. কোন অস্তিত্ব নাই।

Automatic type conversion

see w3school automatic type conversion

5 + null // returns 5 because null is converted to 0
"5" + null // returns "5null" because null is converted to "null"
"5" + 2 // returns "52" because 2 is converted to "2"
"5" - 2 // returns 3 because "5" is converted to 5
"5" * "2" // returns 10 because "5" and "2" are converted to 5 and 2
